def :定義一個東西

print() 映出這個東西 用法 print(‘hello world’)

type 資料的型態

用法: a = 123

               type(a)

結果顯示 init

  init  整數

  str  字串

  float 浮點數

(字串不能直接跟數值相加)

練習:

a = 0
b = 234
c = '123'
print(type(a)) #顯示a的型態
print(a+b) #顯示a+b
#數值與字串相加 需經過轉換
print(a+int(c)) 
print(str(b)+c)#字串相加結果

if-else用法

> 大於

< 小於

>= 大於等於

<= 小於等於

== 等於

!== 不等於

true 真的

false 假的

and 且

or 或

not 非

練習

hp = 80 #生命值80
if hp > 60: #hp大於60時
   print('繼續冒險')
elif hp < 40: #hp小於40時
   print('回家補給')
else: #hp介於60~40之間 #不可放條件
   print('打打小怪就好')

for X in range(a,b) 迴圈  X的迴圈為a到b-1   

for X in range(b) 從頭到b-1

練習

 for X in range(1,10) : #X迴圈1~9
    for Y in range(1,10) : #Y迴圈1~9 
       print(X*Y,end=' ') #end=' '結束時空格
    print(end='\n') #自動換下行
