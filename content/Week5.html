type 資料的型態 用法: a = 123

  init  整數

  str  字串

(字串不能直接跟數值相加)

練習:

a = 0
b = 234
c = '123'
print(type(a)) #顯示a的型態
print(a+b) #顯示a+b
#數值與字串相加 需經過轉換
print(a+int(c)) 
print(str(b)+c)#字串相加結果
